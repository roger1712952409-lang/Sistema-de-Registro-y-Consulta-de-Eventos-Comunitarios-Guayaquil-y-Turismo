# Sistema-de-Registro-y-Consulta-de-Eventos-Comunitarios-Guayaquil-y-Turismo
El Sistema de Registro y Consulta de Eventos Comunitarios Guayaquil y Turismo es una aplicación diseñada para facilitar el registro, administración y consulta de eventos comunitarios, culturales, turísticos y recreativos que se desarrollan en la ciudad de Guayaquil.
